Human Computer Interaction

This is Final Year Project of Group 36 of Rajiv Gandhi Institute of Technology made by Vishal Yadav, Samar Vishwakarma, Alok Yadav and Vaibhav Vijaywargiya under the guidance of Dr. Sharmila Rathod.

This is the project which will be useful for the persons with disabilities especially for deaf people where they can learn and communicate with others via socket io using their hand gestures. These gestures are based on American Sign, as according to the survey 3 out off 1000 people are deaf in US, hence we have used American Sign and also it is globally accepted Sign Language.

This Project will be very much helpful for the deaf people.

Tech Stack Used

Machine Learning is used for Building up the base for the project. Machine learning is a field of inquiry devoted to understanding and building methods that 'learn', that is, methods that leverage data to improve performance on some set of tasks.

MediaPipe offers open source cross-platform, customizable ML solutions for live and streaming media. Unlike power-hungry machine learning Frameworks, MediaPipe requires minimal resources. It is so tiny and efficient to work with it.

Flask is a web application framework written in Python. Armin Ronacher, who leads an international group of Python enthusiasts named Pocco, develops it. Flask is based on Werkzeug WSGI toolkit and Jinja2 template engine. Both are Pocco projects.

HTML is the standard markup language for Web pages, which is used for making a skeleton for the websites which can be further design by CSS and can be provided some functionality using Javascript.

CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed. This is used to attract user and also helps users to interact easily with the websites.

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web, with the help of Javascript we had provided functionality to the website basically it works as a brain of website.

Socket.IO is an event-driven library for real-time web applications. It enables real-time, bi-directional communication between web clients and servers. It consists of two parts: a client-side library that runs in the browser, and a server-side library for backend.
